# MCP Strategy

## Principle

MCP should be:

```txt
minimal
purpose-driven
```

Avoid MCP explosion.

---

## v1 MCP Scope

Only essential integrations.

Candidate MCPs:

```txt
filesystem
git
docker
terminal
github
```

Optional future:

```txt
jira
notion
postgres
redis
exchange API
n8n
```

---

## MCP Security

Every MCP must follow:

```txt
least privilege
explicit scope
approval-aware
```
