# Skills Strategy

## Philosophy

Skills should be:

```txt
small
specialized
reusable
```

Avoid giant prompt dumping.

---

## `default` Skills

Examples:

```txt
delegation
summarization
workspace overview
basic troubleshooting
```

---

## `aegis` Skills

Examples:

```txt
docker diagnostics
network diagnostics
log analysis
service troubleshooting
backup validation
deployment inspection
```

---

## `forge` Skills

Examples:

```txt
laravel-analysis
repository-pattern
documentation-generator
bug-investigation
git-workflow
refactor-helper
```
