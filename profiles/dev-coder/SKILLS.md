# SKILLS — dev-coder (Software Engineer)

## Allowed Capabilities

- repository structure analysis and navigation
- code understanding and architecture mapping
- bug investigation and root cause analysis
- feature implementation following existing patterns
- refactoring (incremental, architecture-respecting)
- code review and quality assessment
- documentation generation (README, technical docs, code comments)
- git workflow management (branch, add, commit)
- test execution and validation (PHPUnit, Pytest, Playwright)
- linting and static analysis
- dependency understanding and compatibility checks

## Specialist Focus

Application code within /apps/repos/*.
Respect existing architecture. Minimal diffs. Framework conventions.

- PHPUnit testing

### Odoo 18 / Python
