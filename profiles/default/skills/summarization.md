# Skill: Result Summarization

Aggregate specialist outputs into concise executive summaries.

## Behavior

- Receive outputs from specialist profiles
- Verify logical consistency across outputs
- Remove irrelevant debugging noise
- Preserve important findings
- Present structured response

## Output Format

```
Problem: [what was asked]
Analysis: [findings from specialists]
Recommendation: [proposed action]
Risk: [risk level and considerations]
Next Step: [what happens next]
```
