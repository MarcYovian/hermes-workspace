## Data Collection

1. **Uptime**: 
   - `uptime -p` for human-readable format
   - Convert to Indonesian: "up 15 days, 3 hours" → "15 hari, 3 jam"

2. **CPU**: 
   - Get load averages: `cat /proc/loadavg | awk '{print $1, $2, $3}'`
   - Get Idle percentage: `top -bn1 | grep "Cpu(s)" | awk '{print $8}' | sed 's/%id,//'`
   - Get Steal percentage: `top -bn1 | grep "Cpu(s)" | awk '{print $16}' | sed 's/%st//'`
   - Format: "Load avg 0.96 / 1.02 / 0.88 — Idle ~95.2% (Steal ~0.1%)"

3. **RAM**: 
   - `free -h | grep Mem | awk '{print $3 " / " $2}'`
   - Get percentage: `free | grep Mem | awk '{printf "%.0f", $3/$2 * 100}'`
   - Format: "3.2Gi / 7.7Gi (42%)"

4. **Disk**: 
   - `df -h / | tail -1 | awk '{print $3 " / " $2 " (" $5 ")"}'`
   - Format: "28G / 49G (60%)"

5. **Docker**: 
   - Check if installed: `command -v docker`
   - Running count: `docker ps --format '{{.Names}}' 2>/dev/null`
   - Total count: `docker ps -a --format '{{.Names}}' 2>/dev/null`
   - Get container details: `docker ps -a --format '{{.Names}}\t{{.Status}}' 2>/dev/null`
   - Parse status to separate running vs stopped
   - For stopped containers, extract exit code and time: "Exited (0) 2 hours ago"
   - Format:
     ```
     🐋 **Docker:** 8/10 running
     - ✅ nginx, postgres, redis, api — semua aktif
     - ⏹️ worker — Exited(137) 2 jam lalu (OOM killed)
     ```

6. **Failed Services**: 
   - `systemctl --failed --no-pager --no-legend | wc -l`
   - If 0: "None"
   - If > 0: list service names

7. **Pending Updates**: 
   - Debian/Ubuntu: `apt list --upgradable 2>/dev/null | grep -c upgradable`
   - RHEL/CentOS: `yum check-update -q | grep -c "^[a-zA-Z]"`
   - If command unavailable: "N/A"

  - All critical Docker containers running
  
- **🟡 PERHATIAN** (Warning): Minor issues detected
